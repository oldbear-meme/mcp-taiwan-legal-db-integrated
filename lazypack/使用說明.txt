台灣法律資料庫 MCP【整合版】— 公務電腦 全自動安裝懶人包（綠色版・含自我診斷）
======================================================================

【一句話】
  雙擊一個檔，從零到「桌面版能查台灣法條/釋字/判決/判解 + 工程會採購法函釋」
  全自動跑完，之後法規、函釋、程式本身都會自動更新，裝一次就不用再管。

【整合版包含什麼】
  - 原版 8 工具：裁判書搜尋/全文、法規查詢、釋字/憲判字、引用圖譜
  - 增補版 3 工具：簡易案件查詢、法令判解系統（判例/決議/法律座談會/司法解釋）
  - 整合版 2 工具：工程會「政府採購法規解釋函令」查詢（3,600+ 則離線快取）
  - 三層自動更新（伺服器啟動時背景執行，失敗不影響使用）：
      1. 法規清單   — 每週六自動更新（11,700+ 部法規）
      2. 工程會函釋 — 每 7 天自動增量抓新函釋
      3. 程式碼     — 每天檢查 GitHub 新版，下次重啟 Claude 自動生效

【為什麼這版能在公務機上裝起來（內建了所有踩過的坑的修正）】
  1. 自帶免安裝 Python —— 不被「微軟商店假 python 捷徑」卡住。
  2. 用 ZIP 下載程式碼 —— 不需要先裝 git。
  3. 不需管理員權限、不改系統 PATH —— 全部裝在自己的資料夾。
  4. 自動修正 MSIX/商店版「讀錯檔」bug —— 設定同時寫進兩處。
  5. 合併式寫設定 —— 保留 Claude 既有偏好，不覆蓋。
  6. 自我診斷 —— 裝完會測 13 個工具 + 實際查一筆函釋，當場驗證。

【跟舊包的關係】
  - 之前裝過「台灣法律MCP（增補版）」：直接跑本包，會就地升級，設定沿用。
  - 之前裝過獨立「工程會函釋MCP」：直接跑本包，函釋功能已併入整合版，
    舊的 pcc-letters 伺服器設定會自動移除（其資料夾可手動刪除 %USERPROFILE%\mcp-pcc-letters）。

【檔案（全部放同一資料夾，不要分開）】
  1. 安裝_台灣法律MCP整合版.bat   <-- 主角，雙擊這個
  2. setup_integrated.ps1         （安裝主程式，不用打開）
  3. 重新診斷.bat                 （日後若出狀況，雙擊它重新體檢）
  4. diagnose.ps1                 （診斷程式，不用打開）
  5. 使用說明.txt

【唯一前提】
  這台電腦要能連到 github.com 與 pypi.org（下載 Python、程式碼、套件）。
  公務網路若擋外網或需 Proxy，安裝會在「裝套件」那步失敗——別擔心，
  程式會把過程存成桌面記錄，照下方處理。

【安裝步驟】
  1) 五個檔放同一資料夾。
  2) 雙擊「安裝_台灣法律MCP整合版.bat」。
     跳「Windows 已保護你的電腦」->「其他資訊」->「仍要執行」。
  3) 看黑視窗跑完 7 步。最後：
       綠色「安裝與自我診斷全部通過！」= 成功
       紅色 = 多半是連不到 pypi（看下方排錯）
  4) 完全結束 Claude 桌面版（右下角工具列圖示 -> 右鍵 -> Quit），重開。
  5) 設定 -> Developer：taiwan-legal-db 應為 running。
  6) 新開對話試問：
       查民法第 184 條
       查工程會關於機關首長的函釋
       查114年高院法律座談會關於消滅時效的法律問題

【裝完桌面會留下】
  - 整合版MCP安裝診斷報告.txt   （整體結果＋設定檔內容）
  - 整合版MCP套件安裝記錄.txt   （pip 完整過程，排查網路問題用）

【出狀況怎麼辦】
  - 紅色「套件沒裝成功」：把上面兩個桌面檔貼給協助你的人（或 Claude）判讀。
  - Developer 顯示 failed：雙擊「重新診斷.bat」，把桌面產出的
    「整合版MCP重新診斷報告.txt」貼回判讀。
  - 想重跑：可放心重複雙擊主安裝檔，已裝好的會略過，只補沒完成的。

【自動更新說明】
  - 全部在 Claude 啟動 MCP 時於背景進行，不用手動操作。
  - 程式碼更新「下次重啟 Claude」才生效；平常重開 Claude 就等於完成更新。
  - 不想要程式碼自動更新：設環境變數 TWLEGAL_SELF_UPDATE=0。

【提醒】
  - 只有「Claude 桌面版」用得到；手機 App、網頁版查不到屬正常。
  - 查到的是政府公開庫的快取，正式定稿前請回官網核對：
      法規 law.moj.gov.tw／判決 judgment.judicial.gov.tw／函釋 planpe.pcc.gov.tw
  - 函釋「已停止適用」標記係依內文關鍵字判定，非 100% 精準，引用前請複核。
